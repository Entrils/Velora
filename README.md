# VELORA

Промо-сайт интерьерной студии VELORA на React + Vite. Одностраничный SPA, плавные GSAP-анимации, тёмная и светлая темы, портфолио, отзывы, услуги и интерактивный блок с «живой квартирой».

https://entrils.github.io/Velora/ - демо

## Что реализовано

- полноэкранный hero-блок с акцентной композицией
- блок о студии со статистикой
- портфолио проектов с модальным окном
- секция отзывов с пользовательскими PNG-карточками
- блок услуг
- интерактивная секция «Живая квартира» в облегчённой псевдо-3D реализации
- контактная секция с формой и картой-заглушкой
- переключение тёмной и светлой темы
- пользовательский курсор для desktop
- GSAP-анимации появления и скролла

## Стек

- React 18
- Vite 5
- GSAP
- CSS Modules

## Запуск

```bash
npm install
npm run dev
```

Открыть локальный адрес Vite, обычно `http://localhost:5173`.

## Как редактировать контент

### Тексты

Весь основной текст лежит в `src/assets/siteContent.js`.

Там можно менять:
- статистику
- проекты
- подписи к отзывам
- список услуг

### Изображения проектов

Файлы лежат в `src/assets/projects/`.

### Изображения отзывов

Файлы лежат в `src/assets/reviews/`.
Сейчас проект использует:
- `review1.webp`
- `review2.webp`
- `review3.webp`

Подключение изображений идёт через `src/assets/siteContent.js`.

## Темизация

В проекте есть две темы:
- тёмная
- светлая

Состояние темы сохраняется в `localStorage` по ключу `velora-theme`.
Основные токены вынесены в:
- `src/styles/variables.css`
- `src/styles/global.css`

## Анимации

GSAP используется для:
- entrance-анимаций
- reveal-анимаций секций
- scroll-based эффектов
- лёгкого параллакса в секции отзывов

## Интерактивный блок

Секция «Живая квартира» находится в:
- `src/sections/ConfiguratorSection.jsx`
- `src/components/LivingSpaceCanvas.jsx`
